# @infra/qui-vue-vant

### 介绍

基于 vant4 vue3 的组件库模板，适用于二次封装或业务组件库。

预览地址：https://quliangen.github.io/qui-vue-vant

### 特性

+ 🚀 vue3 + vant4 
+ 🦋 开箱即用直接添加业务组件即可,零配置。
+ 📚 文档、示例、组件库工程、单测，基于 @vant/cli5。
